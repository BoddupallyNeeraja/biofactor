## Summary

All existing image file names already match their product names correctly. No renaming is needed for existing files.

However, when adding new product images, ensure they follow this naming convention:
- Convert product name to lowercase
- Replace spaces with hyphens
- Replace special characters appropriately:
  - "+" → removed or replaced with hyphen
  - "/" → hyphen
  - ":" → hyphen
  - "." → hyphen (for file extensions only)
  - Remove parentheses and other special chars
